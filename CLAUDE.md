# CLAUDE.md

## Project Overview

This is a personal gift web app called **Samhi's Daily Step 1 Card**. It shows one high-yield USMLE Step 1 flashcard per day (from the AnKing v11 deck), with AI-generated fun facts and research links powered by the Anthropic API.

Read the PRD.md for full product requirements before building anything.

## Setup Instructions

### 1. Install dependencies
```bash
npm install
```

### 2. Extract cards from AnKing deck

Place the AnKing v11 `.apkg` file in the project root, then run:

```bash
pip install -r scripts/requirements.txt
python scripts/extract_cards.py AnKing_v11.apkg public/cards.json
```

This extracts all cards tagged with Step 1 high-yield tags and writes them to `public/cards.json`. The script should filter by tags containing any of:
- `AK_Step1_v11`
- `HY_USMLE`
- High-yield subsets (check actual tag names inside the `.apkg` by querying the `tags` or `notes` table first)

Each card in `cards.json` should have:
```json
{
  "id": "1234567890",
  "front": "<html content of front>",
  "back": "<html content of back>",
  "tags": ["AK_Step1_v11::Cardiology", "HY"]
}
```

### 3. Add API key
```bash
cp .env.example .env
# Add your Anthropic API key to .env
```

`.env.example`:
```
VITE_ANTHROPIC_API_KEY=your_key_here
```

### 4. Run dev server
```bash
npm run dev
```

## Key Implementation Details

### Card Selection (deterministic by date)

Use FNV-1a hash of `YYYY-MM-DD` string, mod by cards array length. Same date = same card on any device.

```js
function fnv1a(str) {
  let hash = 2166136261;
  for (let i = 0; i < str.length; i++) {
    hash ^= str.charCodeAt(i);
    hash = (hash * 16777619) >>> 0;
  }
  return hash;
}

export function getTodaysCard(cards) {
  const today = new Date().toISOString().slice(0, 10); // YYYY-MM-DD
  const index = fnv1a(today) % cards.length;
  return cards[index];
}
```

### Streak Logic (localStorage)

Track streak in localStorage with keys `lastVisitDate` and `streakCount`. On load:
- If `lastVisitDate` is today → streak unchanged
- If `lastVisitDate` is yesterday → streak + 1
- Otherwise → reset streak to 1

### Anthropic API (streaming)

Call the API client-side with streaming enabled. Use `claude-sonnet-4-20250514`. Stream the response into the FunFacts component token by token.

```js
const response = await fetch("https://api.anthropic.com/v1/messages", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": import.meta.env.VITE_ANTHROPIC_API_KEY,
    "anthropic-version": "2023-06-01",
    "anthropic-dangerous-direct-browser-access": "true",
  },
  body: JSON.stringify({
    model: "claude-sonnet-4-20250514",
    max_tokens: 1000,
    stream: true,
    messages: [{ role: "user", content: buildPrompt(card) }],
  }),
});
```

Parse the SSE stream manually — read `data:` lines, parse JSON, accumulate `delta.text` from `content_block_delta` events.

### Fun Facts Prompt

```js
export function buildPrompt(card) {
  return `You are a fun, enthusiastic medical tutor helping a medical student named Samhi study for USMLE Step 1.

Given this flashcard:
FRONT: ${card.front.replace(/<[^>]+>/g, '')}
BACK: ${card.back.replace(/<[^>]+>/g, '')}

Please generate:
1. **Fun Facts** — Two or three genuinely surprising or delightful facts about this topic that go beyond what the card says
2. **Pearl** — One memorable mnemonic or clinical pearl to make this stick
3. **Go Deeper** — One or two links to real articles (PubMed, NEJM, Nature Medicine, etc.) that are actually interesting — only include links you are highly confident exist

Keep it conversational, warm, and a little playful. Use Samhi's name once naturally. No excessive jargon. Use markdown formatting.`;
}
```

### Card HTML Rendering

AnKing cards contain HTML with `<img>` tags (Anki media), inline styles, and sometimes `{{cloze}}` syntax. For v1:
- Render front/back with `dangerouslySetInnerHTML` (safe since this is a personal app with known data)
- Strip cloze markers: replace `{{c\d+::(.+?)}}` with the inner text
- Images won't load (media files not included) — that's acceptable for v1

## Scripts

### `scripts/extract_cards.py`

```python
# Usage: python scripts/extract_cards.py AnKing_v11.apkg public/cards.json
# Requires: pip install beautifulsoup4

import sqlite3, zipfile, json, sys, os, tempfile
from bs4 import BeautifulSoup

HY_TAG_KEYWORDS = ["Step1", "HY", "Physeo", "Pixorize", "OME", "Pathoma"]

def extract(apkg_path, output_path):
    with tempfile.TemporaryDirectory() as tmpdir:
        with zipfile.ZipFile(apkg_path, 'r') as z:
            z.extractall(tmpdir)
        
        db_path = os.path.join(tmpdir, "collection.anki21")
        if not os.path.exists(db_path):
            db_path = os.path.join(tmpdir, "collection.anki2")
        
        conn = sqlite3.connect(db_path)
        cur = conn.cursor()
        
        # Notes table has flds (fields) and tags
        cur.execute("SELECT id, flds, tags FROM notes")
        rows = cur.fetchall()
        
        cards = []
        for note_id, flds, tags in rows:
            # Filter to high-yield Step 1 cards
            if not any(kw in tags for kw in HY_TAG_KEYWORDS):
                continue
            fields = flds.split('\x1f')
            if len(fields) < 2:
                continue
            cards.append({
                "id": str(note_id),
                "front": fields[0],
                "back": fields[1],
                "tags": tags.strip().split()
            })
        
        conn.close()
        
        with open(output_path, 'w') as f:
            json.dump(cards, f)
        
        print(f"Extracted {len(cards)} high-yield cards to {output_path}")

if __name__ == "__main__":
    extract(sys.argv[1], sys.argv[2])
```
